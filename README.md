# workingbeemicrophone
cad data for a replacement bracket

File created in FreeCAD V1.1.0

Based on original plastic part that has a weak point around the screw holes.  

Options:
1) 3D printing - recommend using Nylon for strenght.  If printing parts it may be prudent to modify and strengthen around the screw holes and add more material.  Beward of shrinkage as my first attempt at ordering parts the tolerances were off by enough to be an issue.  Use outputs/MicrophoneBracket.step
2) CNC - JLCCNC can make the parts cheap enough. However shipping and duty made it too costly.  I did not try this option.  Use outputs/MicrophoneBracket.step and CriticalDimentions.pdf.
3) Lasercut - I used sendcutsend and had it cut out of 1/4 stock.  This is smaller than the original but provides space for the friticion washers.  This does require a drill press and relatively precise drilling of the holes and longer machine screws (M3x12).  Use 2d_bracket.svg  This is what I did in the end.  It Al and wont break again.
